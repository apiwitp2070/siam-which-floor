# Siam, Which Floor

Search for correct platform from 2 floors and 4 sides of BTS Siam so you can arrived at your destination correctly.

The website is available in 2 languages, TH and EN.

NOTE: This website is not a route planner app. It's design to only provide some information to look out for while you are in the station. Please visit official BTS Skytrain website for other available info.
